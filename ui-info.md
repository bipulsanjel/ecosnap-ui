# EcoSnap – UI Notes

## 1. Home Screen
- Personalized greeting (with user’s name)
- Featured discovery carousel
- Navigation panel: Home, Snap, Profile

## 2. Recognition Screen
- Central “Tap to Identify” button (opens device camera)
- Secondary options: Select from Gallery
- Uses system camera instead of in-app

## 3. History & Saved Discoveries
- Scrollable cards with previously identified species
- Tap card → view detail

## 4. Profile
- Shows badges, streaks, and app preferences

### Notes:
- Designed with clarity, calmness, and accessibility in mind.
- Fully responsive layouts prepared for mobile form factor.
